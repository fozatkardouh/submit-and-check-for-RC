# What am I?
This is submitRC, with which you can submit solutions to problems for RefugeesCodeAT.

# Usage

```
submitRC problem
```

# How to install

```
Install pip first, then, run this command in the terminal:
sudo -H pip install -e .
```

# How to uninstall

```
sudo -H pip uninstall submitRC
```
